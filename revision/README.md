# Revision Workspace

This folder is a separate content-development area for `The Method of Artificial Systems`.

It does not replace the manuscript source in `source/`. The goal here is to clarify the argument before changing packaging or build layout.

Files:

- `content-review.md`: editorial assessment of the current manuscript and its highest-priority content issues.
- `proposed-structure.md`: a cleaner structure for the work with section-level intent.
- `opening-draft.md`: a rewritten abstract and introduction in a more current, focused voice.

Recommended use:

1. Review `content-review.md` to decide what kind of book/thesis/manifesto this wants to be.
2. Use `proposed-structure.md` as the target shape for the next draft.
3. Pull language from `opening-draft.md` into the LyX source only after the direction feels right.
